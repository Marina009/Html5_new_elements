Графика

Вступление

В HTML5 появилась возможность рисовать графику на веб-странице, используя элементы HTML, вместо того, чтобы зависеть от изображений или сторонних компонентов, таких как Flash. И всё это благодаря двум добавленным графическим элементам: Canvas и SVG.

Перейдём же теперь к их рассмотрению!

Тег <canvas>

Синтаксис:

<canvas>...<canvas/>


О теге:

Этот тег можно назвать одним из самых гибких среди новых тегов HTML, ведь он стал новой основой для веб-игр и других интерактивных компонентов.

<Canvas>, как и его имя, представляет собой чистый лист, двумерный холст, на котором создаются растровые изображения и различные формы, а управлять ими необходимо с помощью кода JavaScript. А созданная данным элементом графика уже сочетается и дополняет остальную часть DOM (объектную модель документа).

Рисунок на холсте HTML, сгенерированный JavaScript может быть преобразован в PNG, и пользователь может загрузить это изображение.


Разрешённый контент тега:

Прозрачный контент (то есть элементы <a>, <audio>, <canvas>, <del>, <ins>, <map>, <object> и <video>, которые наследуют тип содержимого родительского элемента), фразообразующий или потоковый контент.

Специальные атрибуты
Атрибут Значение Описание
width пиксели Задаёт высоту области холста.
height пиксели Задаёт ширину области холста.

Пример применения тега

            <canvas id="canvas" width="500" height="180"></canvas>
<script> const canvas = document.getElementById("canvas");
           const ctx = canvas.getContext("2d");
           ctx.fillStyle = 'purple';
           ctx.shadowOffsetX = 5;
           ctx.shadowOffsetY = 5;
           ctx.shadowBlur    = 5;
           ctx.shadowColor   = 'rgba(204, 204, 204, 0.9)';
           ctx.fillRect(50,5,60,150);</script>

Тег <svg>

Синтаксис:

<svg>...</svg>


О теге:

Расшифровывается, как Масштабируемая векторная графика (Scalable vector graphics) и, соответственно, используется для создания двухмерной векторной графики. Представляет собой формат векторного изображения на основе XML для обеспечения интерактивности.

Как и HTML, SVG имеет DOM (объектную модель документа). SVG, как и HTML поддерживает обработчики событий. А это значит, что, используя эту технологию, можно реализовывать различные интерактивные действия (например, эффекты наведения). Также изображения SVG могут содержать гиперссылки на другие документы.

SVG можно изменять как из CSS, используя таблицы стилей, так и, используя JavaScript. Содержимое SVG можно анимировать с помощью встроенных элементов анимации.


Разрешённый контент тега:

Svg-элементы, то есть элементы анимации, различные фигуры (линия, круг, прямоугольник, эллипс, полигон, полилиния), структурные элементы, градиенты.

Пример применения тега

            <svg height="100" width="120">
    <circle cx="80" cy="50" r="40" fill="turquoise"/>
</svg>
