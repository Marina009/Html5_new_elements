Интерактивность

Вступление

Интерактивные элементы предназначены для взаимодействия с пользователем. Некоторые из таких элеметов позволяют создавать виджеты, то есть определённые элементы взаимодействия в графическом пользовательском интерфейсе, например, кнопка или полоса прокрутки.

Настало время перейти к новым интерактивным элементам, таким как details, summary и dialog.

Тег <details>

Синтаксис:

<details>...</details>


О теге:

Указывает дополнительную информацию, которую можно скрыть или показать в специальном блоке по требованию пользователя, с помощью клика. Также такой блок ещё называют "аккордеоном".


Разрешённый контент тега:

Единственный элемент summary, за которым следует потоковый контент.

Специальные атрибуты
Атрибут Значение Описание
open open Указывает, что содержимое тега изначально должно быть показано в развернутом виде.

Пример применения тега

            <details>
    <summary>Что означает элемент details в HTML?</summary>
    <p>Это блок с раскрывающимся контентом.</p>
</details>

Тег <summary>

Синтаксис:

<summary>...</summary>


О теге:

Определяет видимый заголовок для элемента <details>, то есть это его легенда.
На заголовок можно нажимать, чтобы просмотреть или скрыть информацию. Данный элемент необязателен — без него в качестве легенды будет написан текст «Подробнее».

Пример использования этого тега представлен выше, вместе с тегом <details> .


Разрешённый контент тега:

Фразообразующий контент.

Тег <dialog>

Синтаксис:

<dialog>...</dialog>


О теге:

Используется для создания диалогового окна или модального окна на веб-странице, с которым пользователь может взаимодействовать для выполнения определённой задачи. Когда dialog открыт, остальная часть страницы неактивна.


Разрешённый контент тега:

Потоковый контент.

Пример применения тега с использованием небольшого кода JavaScript

            <div>
    <dialog id="myFirstDialog"></dialog>
    <p>Добро пожаловать на сайт!</p>
    <button id="hide">Закрыть</button>
    </dialog>
    <button id="show">Открыть</button>
</div>
    <script type="text/JavaScript">
    (function() {  
        var dialog = document.getElementById('myFirstDialog');  
        document.getElementById('show').onclick = function() {  
            dialog.show();  
        };  
        document.getElementById('hide').onclick = function() {  
            dialog.close();  
        };  
    })(); 
    </script>
