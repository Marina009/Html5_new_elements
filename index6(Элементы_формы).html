Элементы формы

Вступление

Формы в Интернете — это очень полезная вещь, ведь именно с помощью них принимаются пользовательские данные, например, отзывы, заказы, различные голосования. И чтобы принять пользовательский ввод на HTML-странице, необходимо использовать несколько элементов формы. В HTML5 добавлено пять таких элементов, а именно: meter, progress, datalist, output и keygen. Настало время их рассмотреть поближе.

Тег <meter>

Синтаксис:

<meter>...</meter>


О теге:

Используется для графического представления числового значения (скалярного или дробного) в заданном диапазоне.
Элемент <meter> не должен использоваться для обозначения индикатора прогресса. В таких случаях используется элемент <progress>, который рассмотрен чуть ниже .


Разрешённый контент тега:

Фразообразующий контент. Не может включать другие элементы meter.

Специальные атрибуты
Атрибут Значение Описание
value число Является обязательным атрибутом. Указывает текущее числовое значение, которое должно входить в пределы диапазона, определяемого атрибутами min и max. По умолчанию равно 0.
form id-формы Устанавливает связь между формой и элементом <meter>.
high число Определяет диапазон, при достижении которого значение считается высоким. По умолчанию равно значению атрибута max.
low число Определяет диапазон, при достижении которого значение считается низким. По умолчанию равно значению атрибута min.
max число Обозначает максимальное значение диапазона. Если атрибут не указан, то максимальное значение будет равно 1.
min число Указывает минимальное значение диапазона. Если атрибут не указан, то минимальное значение будет равно 0.
optimum число Определяет оптимальное значение, которое должно входить в пределы диапазона, определяемого атрибутами min и max.

Пример применения тега

            <p>Объём вашей свободной памяти составляет: 
    <meter min="0" max="64" value="40">40 из 64 Гб<meter>
</p>

Тег <progress>

Синтаксис:

<progress>...</progress>


О теге:

Визуально представляет ход выполнения задачи (например, процесс загрузки) и отображает, какой процент задачи уже выполнен. Обычно выглядит как прогресс-бар. Изменения в индикаторе прогресса производятся с помощью JavaScript.
Данный элемент стоит использовать именно для вывода информации о процессе, который должен завершиться. А Для вывода количества, например, места на диске лучше подойдёт элемент meter, который рассмотрен чуть выше .


Разрешённый контент тега:

Фразообразующий контент. Не может включать элементы progress.

Специальные атрибуты
Атрибут Значение Описание
max число Определяет общий объем работы, необходимый для выполнения задачи. Значением атрибута должно быть положительное число. По умолчанию 1.
value число Указывает, какая часть задачи выполнена.

Пример применения тега

            <h3>Прогресс выполнения  задачи</h3>
<p>Прогресс: <progress id="task" value="0" max="100"><span>0</span>%</progress></p>
<script type="text/javascript">
    var i = 0;
    var progressBar = document.getElementById("task");
    function countNumbers(){
        if(i < 100){
            i = i + 1;
            progressBar.value = i;
        }
        setTimeout("countNumbers()", 100);
    }
    countNumbers();
</script>

Тег <datalist>

Синтаксис:

<datalist>...</datalist>


О теге:

Указывает список предопределённых вариантов элемента input. Эти варианты можно выбрать с помощью раскрывающегося меню или с помощью автозаполнения, когда пользователь начинает вводить один из них. Этот элемент используют для быстрого выбора пользователем поля ввода.


Разрешённый контент тега:

Фразообразующий контент, или несколько элементов option.

Пример применения тега

            <label for="favourite-color">Выберите ваш любимый цвет:</label>
<input list="colors" id="favourite-color" name="favourite-color">
<datalist id="colors">
    <option value="Красный">
    <option value="Оранжевый">
    <option value="Жёлтый">
    <option value="Зелёный">
    <option value="Голубой">
</datalist>

Тег <output>

Синтаксис:

<output>...</output>


О теге:

Представляет результат вычислений пользователя. Обычно всё расчитывается с помощью скриптов. Лучше использовать именно этот тег, если нужно показать результат. К примеру, если форма представлена в виде калькулятора, тогда в <output> выводится результат вычислений.


Разрешённый контент тега:

Фразообразующий контент.

Специальные атрибуты
Атрибут Значение Описание
for id-элемента Определяет связь между результатом вычисления и элементами, используемыми при вычислении.
form id-формы Обозначает форму, которой принадлежит элемент output.
name текст Указывает имя для элемента управления формы, используемому при передаче данных формы.

Пример применения тега

            <form oninput="myCalc.value = parseInt(a.value) * parseInt(b.value)">
<input type="number" id="a"> * <input type="number" id="b"> = 
<output name="myCalc" for="a b"></output>
</form>

Тег <keygen>

Синтаксис:

<keygen/>


О теге:

Используется как часть элемента form для того, чтобы генерировать пару ключей (закрытый и открытый). Эти ключи используются для шифрования и расшифровки данных. Когда пользователь отправляет форму, открытый ключ отправляется на сервер, а закрытый – хранится локально.


Разрешённый контент тега:

Пустой, так как этот элемент включает только начальный тег и не может содержать какие-либо контенты.

Пример применения тега

            <form>Никнейм пользователя: <input type="text" name="user_name"><br>
    Шифрование: <keygen name="secure"><input type="submit">
</form>
